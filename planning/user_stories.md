# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

User: A registered user who can browse events, RSVP to events, and will choose to sign up to create/manage their own events.

## Draft User Stories

1. As a platform user, I want to create an account so that I can save my interests and start discovering events tailored to me.
2. As an event-goer, I want to browse upcoming events so that I can discover things happening near me that match my interests.
3. As an event-goer, I want to filter events by category so that I don't have to sift through events I'm not interested in.
4. As an event-goer, I want to search events by keyword so that I can quickly find a specific event I already have in mind.
5. As an event-goer, I want to view full details of an event (time, location, description) so that I can confidently decide whether it fits my schedule and interests.
6. As an event-goer, I want to RSVP to an event so that I have a spot secured and the organizer can plan for my attendance.
7. As an event-goer, I want to cancel my RSVP so that my spot opens up for someone else if my plans change.
8. As an event-goer, I want to see how many people are attending so that I can gauge its size and decide if it's the right fit for me.
9. As an organizer, I want to create an event with key details (name, description, date, location, category) so that the right people can find it and show up.
10. As an organizer, I want to edit my event's details so that attendees always have accurate, up-to-date information.
11. As an organizer, I want to delete an event I created so that attendees are promptly notified it's no longer happening and aren't left showing up to nothing.
13. As an organizer, I want to see my event's attendee list so that I can prepare adequately (seating, supplies, capacity).
14. As a platform user, I want a profile page showing both the events I'm organizing and the ones I've RSVP'd to so that I can manage my commitments in one place without hunting across the app.
